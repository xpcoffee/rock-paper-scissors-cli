# rock-paper-scissors-cli

A rock-paper-scissors command-line game.

## Pre-requisites

* Java 21 SDK

## Running

Launch the application:
```bash
./gradlew bootRun
```

## Usage

via Gradle: 
```bash
./gradlew -q --console plain run
```

Running in IntelliJ: double-click `run` from Gradle tasks.
